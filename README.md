# 💰 Expense Tracker - MERN Stack

A beautiful, user-friendly personal expense tracker built with the MERN stack (MongoDB, Express, React, Node.js). Tailored for Indian users with INR (₹) currency and India-specific categories.

## 🎯 Features

- 📊 **Interactive Dashboard** with real-time statistics
- 📈 **Visual Analytics** using charts (Line charts for trends, Pie charts for categories)
- 💸 **Transaction Management** with income and expense tracking
- 📱 **Responsive Design** - Works on desktop, tablet, and mobile
- 🎨 **Premium Dark Theme** with glassmorphism effects
- 🇮🇳 **Indian Context** - INR currency, Indian categories (D-Mart, Ola, etc.)

## 🛠️ Tech Stack

### Frontend
- **React** (Vite) - Fast and modern React setup
- **Lucide React** - Beautiful icon library
- **Recharts** - Interactive charts and data visualization
- **CSS3** - Custom design system with CSS variables

### Backend (Coming Soon)
- **Node.js** with Express.js
- **MongoDB** for database
- **JWT** for authentication
- **RESTful API** architecture

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Jaiminkansagara1327/Expense_tracker.git
   cd Expense_tracker
   ```

2. **Install frontend dependencies**
   ```bash
   cd frontend
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173`
